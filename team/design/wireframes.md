```
│                                     │

│                                     │
│  文字起こし・選定・字幕・投稿文まで全自動  │  サブコピー: text-gray-600
│                                     │
│  ─────────────────────────────────  │
│                                     │
│  ┌─────────────────────────────┐   │  カード: rounded-xl shadow p-6
│  │  YouTube の URL              │   │
│  │  ┌─────────────────────┐   │   │  Label + Input
│  │  │ https://youtube.com/... │   │   │  inputmode="url"
│  │  └─────────────────────┘   │   │  placeholder例
│  │  ← エラー時: 赤文字でフィールド直下 │   │
│  │                              │   │
│  │  メールアドレス               │   │
│  │  ┌─────────────────────┐   │   │  Label + Input
│  │  │ you@example.com      │   │   │  inputmode="email" type="email"
│  │  └─────────────────────┘   │   │
│  │  ← エラー時: 赤文字でフィールド直下 │   │

│  │  │✓│ 使用します。著作権の確認は   │   │  未チェック時: ボタン非活性
│  │  └─┘ 自己責任で行います。        │   │
│  │       利用規約を読む →         │   │  利用規約リンク
│  │                              │   │
│  │  ┌─────────────────────┐   │   │  送信ボタン
│  │  │    切り抜きを作る →    │   │   │  bg-primary text-white
│  │  └─────────────────────┘   │   │  同意チェックOFF → opacity-50 disabled

│                                     │
│  登録不要 · 無料でお試しいただけます      │  補足テキスト: text-sm text-gray-500
│                                     │
└─────────────────────────────────────┘
```

**状態定義**：
- `idle`：初期状態
- `submitting`：送信中（ボタンスピナー表示、再送信不可）
- `error_url`：URL形式不正（赤枠 + エラーテキスト）
- `error_email`：メール形式不正（同上）
- `error_terms`：同意未チェックで送信試みた場合（チェックボックス強調）
- `error_429`：キュー満杯（フォーム上部にバナー）
