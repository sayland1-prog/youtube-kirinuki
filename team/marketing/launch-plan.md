# ローンチ計画 — youtube-kirinuki

---

## フェーズ別ロードマップ

| フェーズ | タイミング | 施策 |
|---------|----------|------|
| **クローズドβ** | デプロイ直後 | 身内・知人の切り抜き師5〜10人に試してもらう。フィードバック収集 |
| **ソフトローンチ** | β安定後 | X(旧Twitter)で初告知。切り抜き界隈に発信 |
| **本ローンチ** | 1〜2週間後 | 実例動画付きで本格告知。コミュニティ投稿 |

---

## 告知チャネル

### X（旧Twitter）優先チャネル
- ハッシュタグ: `#切り抜き` `#ショート動画` `#YouTube切り抜き` `#副業`
- フォロー推奨アカウント（要調査）: 切り抜き許可を出している配信者・切り抜き師コミュニティ

### その他チャネル
- YouTube切り抜き師コミュニティ（Discord・掲示板・Xスペース）
- note（使い方ガイド記事）
- 切り抜き許可を出している配信者に直接連絡（導入事例として協力依頼）

## 告知文テンプレ

### X投稿用（初告知）
```
【新サービス】YouTubeのURLを貼るだけで切り抜きショートが自動生成されます。

✅ 文字起こし・ハイライト選定・字幕・投稿文まで全自動
✅ 30分でメールに届く
✅ 登録不要・無料で試せる

切り抜き許可のある動画限定です。ぜひ試してみてください👇
[URL]

#切り抜き #ショート動画
```

### X投稿用（デモ動画付き）
```
URLを貼るだけで切り抜き動画を自動生成するサービスを作りました。

→ 実際に生成した動画がこちら（許可済み素材を使用）
[デモ動画/GIF]

登録不要・無料 → [URL]

#YouTube切り抜き #自動化 #副業
```

---

## デモ素材の調達方針

**必ず自分が権利を持つ、または許可を得た素材のみ使用する。**

調達ステップ：
1. 切り抜き許可を出している配信者に事前連絡し、デモ利用の許可を取る
2. 許可確認を記録（DM/メールのスクリーンショット等）
3. 完成デモをX/noteに投稿する際は「○○さんの許可済み素材を使用」とクレジット

## KPI ファネルと計測設計

```
LP訪問 → URL送信（着手）→ 生成完了 → ダウンロード → 再利用（リピート）
```

### 計測イベント（ENGへの依頼）

| イベント名 | タイミング | 優先度 |
|-----------|----------|--------|
| `page_view` | トップ訪問時 | ★必須 |
| `job_submit` | URL送信成功時 | ★必須 |
| `job_complete` | 生成完了時 | ★必須 |
| `download_click` | DLボタンクリック時 | ★必須 |
| `job_failed` | 処理失敗時 | ★必須 |
| `return_visit` | 同メールアドレスが再利用 | 推奨 |

計測ツール: Google Analytics 4（GA4）または Plausible（プライバシー重視）→ 要オーナー決定

### 最重要指標（初期）
- **URL送信率**（LP訪問 → 送信）：初期目標 20%以上
- **完了率**（送信 → 生成完了）：90%以上（処理品質指標）
- **DL率**（完了 → DLボタンクリック）：70%以上
