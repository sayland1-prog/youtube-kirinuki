# タスク指示書 — マーケティング担当

> 発行：オーケストレーター ／ 指針：`team/marketing/CLAUDE.md` ／ 全体計画：`team/orchestrator/execution-plan.md`

## ブリーフ（依頼の要旨）

- **【背景】** North Star「URL入れるだけ」を市場に届ける土台作り。P0でポジショニングを固めないと、デザイン・LP・料金が全部ブレる。
- **【成果物】** P0：ポジショニング1枚＋一次メッセージ。P1：LP構成・コピー、計測要件。P3：告知・規約文言・料金案。
- **【制約】** 要件§2。**許可済み素材前提**の訴求（無断転載を煽らない）。誇大表現禁止。競合情報は必ず一次確認。**金額の最終決定はオーナー**。
- **【完了条件】** 各成果物が他担当に渡せる粒度＋オーナー確認を通過。
- **【依存】** 先行：なし（即着手可）。後続：デザイン(LP)、ENG(計測実装・料金根拠)。

---

## P0（SP1）— 即着手・最優先
- [ ] ターゲット ICP 確定（第一＝**切り抜き許可を得ている切り抜き師**）
- [ ] ポジショニング1枚（差別化3軸：完全自動×最小操作／日本語特化／投稿一式が揃う）
- [ ] 一次メッセージ＆トーン定義（例「URLを貼るだけ。30分で完成。」）
- [ ] **著作権の伝え方ガイドライン**（炎上回避の文言ルール）→ デザイン/規約に共有
- 🚩 提出先：デザイン（LP訴求軸として）／統括（M1レビュー）

## P1（SP2）
- [ ] LP構成案＋セクション別コピー（ヒーロー/価値/使い方/FAQ/CTA）→ デザイン
- [ ] KPIファネル定義（訪問→送信→完了→DL→リピート）＋計測イベント要件 → ENG
- [ ] 競合一次調査（Opus Clip / 2short.ai / Vrew 等の機能・価格を**ファクトで**）

## P3（SP5）
- [ ] 料金プラン案＋根拠（← ENGの実コスト）※金額決定はオーナー
- [ ] ローンチ告知文＋チャネルリスト（X／切り抜きコミュニティ等）
- [ ] 利用規約の**訴求面**文言案（法務最終はオーナー）
- [ ] デモ素材の用意（**自分が権利を持つ/許可済み**のものに限る）

---

## Definition of Done
- ポジショニングがデザイン/ENGの判断基準として機能する
- すべての文言が「許可済み前提・誇大表現なし」を満たす
- 料金案に実コスト根拠が添う
