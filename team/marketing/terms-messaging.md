## 著作権の伝え方ガイドライン

### 基本方針
- このサービスは **許可チェックを行わない**（技術的に確認できない）
- → 利用規約への同意でユーザーの自己責任とするモデル
- マーケ・UI文言はこの前提を踏み外さない

### 言ってよいこと ✅
- 「切り抜き許可のある動画にご利用ください」
- 「著作権の確認はユーザーご自身の責任です」
- 「配信者が定めるガイドラインに従ってください」
- 「切り抜き許可をもらったら、URLを入れるだけ」

### 言ってはいけないこと ❌
- 「どんな動画でも切り抜ける」（無断転載を示唆）
- 「著作権は気にしなくてOK」
- 「バレない」「グレーゾーン」
- 「収益化できる」（配信者の条件による）
- 「絶対安全」「問題なし」

## 利用規約の訴求面 文言案

### 同意チェックボックスの文言（UI用）
```
切り抜き許可のある動画にのみ使用します。
著作権および切り抜きガイドラインの確認は自己責任で行います。
利用規約に同意する
```

### 利用規約 著作権条項（案）
```
第◯条（著作権・コンテンツの取り扱い）

1. 本サービスは、ユーザーが入力した動画URLの著作権状態を確認しません。
2. ユーザーは、本サービスを利用する前に、対象動画の著作権者（配信者等）から
   切り抜き・二次利用に関する許可を得ていることを自ら確認する責任を負います。
3. 著作権者が定める切り抜きガイドライン（クレジット表記、収益化条件等）が
   ある場合は、ユーザーがそれに従う責任を負います。
4. 本サービスの利用により著作権侵害が生じた場合、当社は一切の責任を負いません。
```
> ⚠️ 最終文面は法務担当またはオーナーの確認が必須。上記はあくまでドラフト。

```

YouTubeの動画からショート動画を自動生成する機能を提供しますが、
その動画を使用してよいかどうかの判断はユーザー自身が行う必要があります。

切り抜きを行う前に、必ず以下を確認してください：
・配信者が切り抜きを許可しているか
・収益化・クレジット表記など条件があるか
・配信者の最新ガイドラインを確認しているか

無断転載・著作権侵害への利用は、利用規約により禁止されています。
```
